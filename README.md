1. Створіть клас Node, який представлятиме вузол дерева. Кожен вузол має мати наступні властивості: значення (тип даних може бути будь-яким), посилання на лівого та правого нащадка. Створіть метод, який дозволяє додавати нащадків до вузла.
2. Створіть клас Tree, який представлятиме дерево. Кожне дерево має мати корінь (вузол) та метод, який дозволяє додавати вузли до дерева.
3. Реалізуйте методи preOrder, inOrder та postOrder, які дозволяють обійти дерево в прямому, симетричному та зворотньому порядку відповідно. Кожен з цих методів має використовувати асинхронні потоки.
4. Реалізуйте метод find, який дозволяє знайти вузол з певним значенням в дереві. Цей метод має також використовувати асинхронні потоки.
5. Створіть метод delete, який дозволяє видалити вузол з дерева. Цей метод має також використовувати асинхронні потоки.
6. Реалізуйте метод update, який дозволяє змінити значення вузла в дереві. Цей метод має також використовувати асинхронні потоки.
7. Створіть тестовий клас, який додає вузли до дерева та викликає раніше реалізовані методи.
